# VM Setup and Automation Project

This repository contains scripts and configuration files to automate the setup and management of virtual machines (VMs) for a consistent development environment. The goal is to streamline the initialization of VMs by automating package updates, environment setup, and necessary configurations.

---

## Overview

- VMs are often created from older snapshots and may not be up to date.
- This project automates updating and configuring VMs to ensure consistency.
- Scripts install required packages, set up Python virtual environments, and configure Git.
